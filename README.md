## 快速开始

### 运行环境需求

为了运行此网站代码, 你需要先安装以下程序并运行
+ Node.js
+ Mongodb

### 运行代码

请在Windows/Linux下用命令行进入项目路径后运行以下命令
``` bash
# 安装依赖包, 有可能被墙, 请用淘宝的restry
npm install

# 让代码跑起来
node app.js
```

如果顺利的话, 网站就能运行起来了, 进入`http://localhost:3000`访问网站。
